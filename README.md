# QCF

Qcard Content Format

QCFは、QcardおよびGEDAプロジェクトで利用される知識コンテンツ交換フォーマットです。

## Purpose

QCFは以下を目的とします。

* AIによるコンテンツ生成
* コンテンツ共有
* コンテンツ販売
* 学習データの標準化
* 長期的な知識アーカイブ

## Projects

* GEDA
* Qcard PRO
